# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-17

### Added
- Initial release of E-commerce Product Comparison Tool
- Support for 7 major e-commerce sites:
  - Amazon (Global)
  - Flipkart (India)
  - eBay (Global)
  - Myntra (India)
  - Ajio (India)
  - Walmart (Global)
  - Snapdeal (India)
- Robust scraping with multiple selector fallbacks
- Parallel processing for faster results
- Automatic retry mechanism
- Popup handling
- Multiple demo versions:
  - Console application
  - GUI application (Java Swing)
  - Web-based demo
  - Interactive live demo
- Comprehensive documentation
- Maven build configuration
- Error handling and logging

### Features
- Multi-site product comparison
- Real-time price comparison
- Rating aggregation
- Site-wise organized results
- Configurable scraper service
- Cross-platform compatibility
